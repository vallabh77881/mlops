# Experiment 6: Dockerizing a Web Application

## 📌 Project Overview
This project demonstrates how to containerize a simple web application using Docker within a WSL2 environment.

## 🚀 Features
* Uses **Nginx** as a lightweight web server.
* Automated build process via a **Dockerfile**.
* Port mapping to allow access from the Windows host browser.

## 🛠️ Prerequisites
Before running this, ensure you have:
* **WSL2** installed (Ubuntu/Debian).
* **Docker Desktop** (with WSL2 integration enabled) or Docker Engine installed in WSL.
